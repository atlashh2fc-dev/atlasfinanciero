# Deployment safety

- **Never inspect, deploy to, promote, alias, reassign, or otherwise modify `atlas.geimser.cl` from this repository.** That domain belongs exclusively to the CRM and is outside the scope of Atlas Financiero / Facturación.
- Do not use `atlas.geimser.cl` as a production-verification target for this project.
- Before changing any custom production domain for this repository, obtain the user's explicit confirmation of the exact hostname and target deployment.
